Instrucciones de Configuración y Ejecución

Versiones Requeridas:
- Java 17
- Gradle 8.4

Pasos para la Ejecución:

1. Asegúrate de haber instalado y configurado correctamente las variables de entorno para Java y Gradle en tu sistema.

2. Abre una terminal en la carpeta raíz del proyecto.

3. Ejecuta el siguiente comando para limpiar y ejecutar las pruebas:./gradlew clean test o gradle clean test


Estos pasos deberían limpiar el proyecto y ejecutar las pruebas utilizando las versiones específicas de Java y Gradle mencionadas anteriormente. Asegúrate de revisar la salida de la terminal para cualquier mensaje de error o advertencia que pueda surgir durante la ejecución.

Para ver el reporte generado, ve a la siguiente ruta después de la ejecución de las pruebas:/build/karate-reports/bp.ec.store.html

Este archivo de reporte proporcionará información detallada sobre el resultado de las pruebas. ¡Espero que esto te sea útil!
